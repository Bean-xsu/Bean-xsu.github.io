# Bean的类型

#####  Spring有两种bean，一种叫普通bean，另一种叫工厂bean(FactoryBean)

- <h5>普通bean：</h5>在配置文件中定义bean是什么类型，返回的就是什么类型。

- <h5>工厂bean：</h5>在配置文件中定义的bean的类型可以和返回的类型不一样。
