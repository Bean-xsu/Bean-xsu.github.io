# Vue简介
## 官网

1. [英文网址](https://vuejs.org)
2. [中文网址](https://cn.vuejs.org)

## 介绍与描述

1. 动态构建用户界面的**渐进式**JavaScript框架
2. 作者：尤雨溪

## Vue的特点

1. 遵循**MVVM**模式
2. 编码简洁，体积小，运行效率高，适合移动/PC端开发
3. 本身只关注UI，也可引入其他第三方库开发项目
